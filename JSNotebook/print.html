My JavaScript Notebook: print and println

We learned about print and println; about what they do and their unique properties that differentiate them. We use print and println to put text and numbers onto the screen. We would typically use this type of code to display information that the user might find beneficial. The difference between them however is that if we were to use println, it puts the coding after it into a separate line. If you want text on the same line you would print.
